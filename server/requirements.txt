# Python зависимости для сервера мониторинга
# Все библиотеки входят в стандартную поставку Python 3.8+

# Основные библиотеки (встроенные в Python):
# - socket - для сетевого соединения
# - json - для обмена данными
# - threading - для многопоточности
# - ssl - для шифрования (если потребуется)
# - hashlib - для хеширования и аутентификации
# - time - для работы со временем
# - logging - для логирования
# - datetime - для работы с датой и временем
# - base64 - для кодирования изображений

# Никаких внешних зависимостей не требуется!
# Все используемые библиотеки входят в стандартную поставку Python.
